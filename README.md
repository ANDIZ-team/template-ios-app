# template-ios-app

Andiz 신규 iOS 앱을 시작하기 위한 템플릿 레포.

## 사용법

1. 우측 상단 **`Use this template`** 버튼 클릭
2. 새 레포 이름 입력 (앱 이름 그대로, 예: `TwoPotato`)
3. 새 레포 클론 후 다음 작업:
   - 이 README를 새 앱 README로 교체
   - Xcode 프로젝트 생성 (이름 = 앱 이름)
   - 첫 커밋: `chore({이슈키}): initial app setup`

## 자동 포함되는 것

- `CODEOWNERS` — 두 명 자동 리뷰어 지정
- `.gitignore` — Swift / Xcode / SPM / Fastlane / macOS 표준
- 조직 차원 PR 템플릿 (메타 레포 `.github`에서 자동 적용)
